
## 📖 About – Academic Study Assistant

**Academic Study Assistant** is an AI-powered study companion that helps students and researchers get **precise answers from PDFs and the web** using a **Retrieval-Augmented Generation (RAG)** pipeline.

It can process **uploaded academic documents**, **search the web**, and **deliver concise, context-rich answers with metadata** for easy reference.

---

### ✨ Key Features

1. **📂 PDF Upload & Processing**

   * Upload your research papers, books, or class notes.
   * Automatically **extracts, cleans, and chunks text** for semantic search.
   * Supports **multi-PDF retrieval** for complex queries.

2. **🤖 RAG-Powered Chat**

   * Answers your queries using **LLM reasoning + retrieved chunks**.
   * Maintains **short, meaningful context** for high-quality responses.
   * Supports multiple modes:

     * `study_material` → Searches uploaded PDFs
     * `web_search` → Fetches live results from the internet

3. **🌐 Web Search **

   * Integrates **Google Serper API** for live web search.
   * Retrieves **search snippets** :
   * Ensures every answer can be **cited or explored further**.

4. **🎯 Organized Relevant Chunks**

   * Each response includes:

     * **Context snippets** (short & meaningful)
     * **Metadata** (title + link for references)

---

This assistant is perfect for:

* **Summarizing study material quickly**
* **Researching academic topics** with citations
* **Combining private notes + web knowledge** in a single query
